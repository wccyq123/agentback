# AgentBack

**已部署至vercel，可以在线体验**
- https://agentback-wcyq123-4533-wcys-projects-53f3140d.vercel.app/

**技术栈**
- CopilotKit（React、Runtime）
- LangChain.js、LangGraph
- 向量数据库：MongoDB Atlas
- 大语言模型：DeepSeek
- 向量化模型：text-embedding-v4

**快速开始**
- 安装依赖：`pnpm install`
- 本地开发：`pnpm -F web dev`
- 生产构建：`pnpm -F web build`
- 生产启动：`pnpm -F web start`
